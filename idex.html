1. What is Flask?

Flask is a lightweight Python web framework used to create websites, REST APIs, dashboards and backend applications. Flask gives you the essential web features and allows you to select your own database and extensions.

Frontend

HTML creates structure, CSS creates design, and JavaScript adds browser interaction.

Backend

Flask receives requests, runs Python logic, communicates with the database and returns responses.

Application flow

Browser → Flask Route → Python Logic → PostgreSQL ← HTML / JSON Response ←

3. Your First Flask Application

Create app.py:

from flask import Flask

app = Flask(__name__)

@app.route("/")
def home():
    return "Hello, Flask!"

if __name__ == "__main__":
    app.run(debug=True)

Explanation

Code Meaning
Flask(__name__) Creates the Flask application.
@app.route("/") Connects the home URL to a Python function.
return Sends a response to the browser.
debug=True Automatically reloads during development and shows detailed errors.
Important: Do not use debug mode in production.

4. Routes, Dynamic URLs and HTTP Methods

@app.route("/")
def home():
    return "Home Page"

@app.route("/about")
def about():
    return "About Page"

@app.route("/student/<name>")
def student(name):
    return f"Welcome, {name}"

@app.route("/marks/<int:score>")
def marks(score):
    return {"score": score}

Main HTTP methods

Method Purpose
GET Read or display data.
POST Create or submit data.
PUT / PATCH Update data, commonly used in APIs.
DELETE Delete data, commonly used in APIs.

8. PostgreSQL Database Setup

Create database using pgAdmin

  1. Install PostgreSQL and open pgAdmin.
  2. Create a database named flask_students.
  3. Remember the username, password, host and port.

Connection URL format

postgresql://USERNAME:PASSWORD@HOST:PORT/DATABASE_NAME

Local example:

postgresql://postgres:your_password@localhost:5432/flask_students

Create a .env file

DATABASE_URL=postgresql://postgres:your_password@localhost:5432/flask_students
SECRET_KEY=replace-with-a-long-random-secret
Security: Never upload .env to GitHub. Add it to .gitignore.

9. Flask-SQLAlchemy and Database Models

An ORM lets you work with database tables using Python classes.

import os
from dotenv import load_dotenv
from flask import Flask
from flask_sqlalchemy import SQLAlchemy

load_dotenv()

app = Flask(__name__)
app.config["SQLALCHEMY_DATABASE_URI"] = os.getenv("DATABASE_URL")
app.config["SQLALCHEMY_TRACK_MODIFICATIONS"] = False

db = SQLAlchemy(app)

class Student(db.Model):
    id = db.Column(db.Integer, primary_key=True)
    name = db.Column(db.String(100), nullable=False)
    email = db.Column(db.String(120), unique=True, nullable=False)
    course = db.Column(db.String(100), nullable=False)

    def __repr__(self):
        return f"<Student {self.name}>"

11. Complete Basic Student Management Project

app.py

import os
from dotenv import load_dotenv
from flask import Flask, render_template, request, redirect, url_for, flash
from flask_sqlalchemy import SQLAlchemy
from sqlalchemy.exc import IntegrityError

load_dotenv()

app = Flask(__name__)
app.config["SECRET_KEY"] = os.getenv("SECRET_KEY", "development-secret")
app.config["SQLALCHEMY_DATABASE_URI"] = os.getenv("DATABASE_URL")
app.config["SQLALCHEMY_TRACK_MODIFICATIONS"] = False

db = SQLAlchemy(app)


class Student(db.Model):
    id = db.Column(db.Integer, primary_key=True)
    name = db.Column(db.String(100), nullable=False)
    email = db.Column(db.String(120), unique=True, nullable=False)
    course = db.Column(db.String(100), nullable=False)


@app.route("/")
def home():
    students = Student.query.order_by(Student.id.desc()).all()
    return render_template("index.html", students=students)


@app.route("/students/add", methods=["POST"])
def add_student():
    name = request.form.get("name", "").strip()
    email = request.form.get("email", "").strip().lower()
    course = request.form.get("course", "").strip()

    if not name or not email or not course:
        flash("All fields are required.", "error")
        return redirect(url_for("home"))

    student = Student(name=name, email=email, course=course)

    try:
        db.session.add(student)
        db.session.commit()
        flash("Student added successfully.", "success")
    except IntegrityError:
        db.session.rollback()
        flash("This email already exists.", "error")

    return redirect(url_for("home"))


@app.route("/students/<int:student_id>/edit", methods=["GET", "POST"])
def edit_student(student_id):
    student = db.get_or_404(Student, student_id)

    if request.method == "POST":
        student.name = request.form.get("name", "").strip()
        student.email = request.form.get("email", "").strip().lower()
        student.course = request.form.get("course", "").strip()

        try:
            db.session.commit()
            flash("Student updated successfully.", "success")
            return redirect(url_for("home"))
        except IntegrityError:
            db.session.rollback()
            flash("That email is already being used.", "error")

    return render_template("edit.html", student=student)


@app.route("/students/<int:student_id>/delete", methods=["POST"])
def delete_student(student_id):
    student = db.get_or_404(Student, student_id)
    db.session.delete(student)
    db.session.commit()
    flash("Student deleted successfully.", "success")
    return redirect(url_for("home"))


with app.app_context():
    db.create_all()


if __name__ == "__main__":
    app.run(debug=True)

13. Common Errors and Fixes

Error Likely fix
ModuleNotFoundError: flask Activate the virtual environment and run pip install Flask.
connection refused Start PostgreSQL and check host, port, username and password.
password authentication failed Correct the PostgreSQL password in DATABASE_URL.
TemplateNotFound Place HTML files inside the exact folder name templates.
CSS is not loading Place CSS inside static/css and use url_for('static', ...).
Duplicate email error Handle IntegrityError, rollback the session, and show a message.
Working outside of application context Run database setup code inside with app.app_context():.
